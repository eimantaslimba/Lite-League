# Lite-League

Lightweight Rocket League in Unity

Mechanics Powered by [RocketSim](https://github.com/ZealanL/RocketSim)

[Download Build](https://github.com/eimantaslimba/Lite-League/releases/download/beta/Lite.League.zip)

Keyboard controls

- WASD QE Shift Space
- **5** -> reset map
- **numpad \*** -> boost
- **numpad -** -> jump
